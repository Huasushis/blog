题解

这里是简要的思路。

T1

一眼闵可夫斯基和，参看 [JSOI2018] 战争 ，然后对于速度求到一条边最短距离即可。

T2

容易发现一些规律，直接加减，发现和 c 的历史最低值有关。标注题解的证明有一点复杂，这里提供一种形式化的证明。考虑 0 刻度线上的点表示 a 的值为 0。a 的值为相对 0 刻度线的位置。考虑一次减法，如果减到小于 0，则 0 刻度线要向下移动 k。如此理解，就是那样了。

T3

考虑尾巴加一个进去的改变，的复杂度，然后考虑合并，一样的。然后点分治点分树上面搞就行。写不来。也可以倍增。
